Tecnocluster es un complemento que organiza las entradas y las páginas en cluster de imágenes o conjunto de imágenes.

== Description ==

Puede comunicarse con nosotros a través del sitio web: Click here => [https://amdevelops.com)**

Para usarlo es muy sencillo solo vaya a tecnocluster clic en agregar nuevo y siga las indicaciones,
es sumamente fácil de usar y cualiquier persona lo pordría utilizar, una vez que creó su primer tecnocluster
sus usuarios se lo agradecerán.

== Installation == 
1. Descargue y descomprima el complemento. Sube la carpeta descomprimida a la carpeta wp-contents / o usa tu instalación de plugins de WordPress.
2. Active el complemento desde la página de administración de Complementos de WordPress.
3. Después de la instalación, puede ver "Panel de control" un nuevo menú "Tecnocluster".

== Frequently Asked Questions ==
= How do I use the plugin? =

La respuesta es muy sencilla, tan solo ve al menú de wordpress y busca a tecnocluster, le das agregar nuevo y sigue la 
documentación escrita en el complemento.
